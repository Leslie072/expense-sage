## 🚀 Quick Start

### Prerequisites
- Flutter SDK 3.16.0+
- Dart 3.0+

### Local Development
```bash
# Clone repository
git clone <repository-url>
cd expense_sage

# Install dependencies
flutter pub get

# Run web version
flutter run -d web-server --web-port 8080
```

### Production Deployment
This app is configured for one-click Heroku deployment with Docker.
